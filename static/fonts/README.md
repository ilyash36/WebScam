# Шрифты

## KOT-Eitai Gothic Bold, Century Old Style Std

Шрифты загружаются локально из этой папки (`KOT-EitaiGothicBold.woff2`, `CenturyOldStyleStd.woff2`).
При отсутствии локальных файлов используется CDN OnlineWebFonts.

## DwarvenStonecraftCyrExtended

Для отображения логотипа в шапке сайта необходим шрифт **DwarvenStonecraftCyrExtended**.

**Как добавить:**

1. Скачайте файл `DwarvenStonecraftCyrExtended.otf` с сайта:
   https://myfontlib.com/font/dwarven-stonecraft-font

2. Поместите файл в эту папку: `static/fonts/DwarvenStonecraftCyrExtended.otf`

3. Перезагрузите страницу сайта

Без этого файла логотип будет отображаться шрифтом KOT-Eitai Gothic Bold (резервный вариант).
